# Security Policy

## Supported Versions

Only the latest `main` branch is supported for security patches.

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |
| < v0.x  | :x:                |

## Reporting a Vulnerability

We take the security of Wapex seriously. If you believe you have found a security vulnerability in this project, please **do not report it through a public issue or pull request**.

Instead, please report it via one of the following methods:

1.  **Direct Email**: [INSERT SECURITY CONTACT EMAIL]
2.  **Private GitHub Disclosure**: Use the "Report a vulnerability" button on the security tab of this repository.

Please provide a detailed description of the vulnerability, including:
*   Potential impact
*   Step-by-step instructions to reproduce
*   Any suggested fixes or mitigations

We will acknowledge receipt within 48 hours and work with you to resolve the issue as quickly as possible. We will also provide credit for your discovery if you'd like.

Thank you for helping keep Wapex secure!
